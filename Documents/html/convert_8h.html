Converts an object mesh in STL format into its equivalent AMF formatted vertices and triangles. STLs represent a 3D object as individual facets and vertices, with each vertex distinct from all others ("shared" vertices are only microns distance apart). weld combines vertices that are effectively the same point in space into the same vertex and re-indexes all the STL triangles such that the resulting list of indexed vertices and triangles represent the same model except condensed as necessary.

Parameters:
unwelded_vertices a QList of FAHVector pointers that refer to the STL vertices
unwelded_facets a QList of STLFacet pointers that refer to STL-indexed triangles
welded_vertices a QList of FAHVector pointers that will contain the AMF-welded vertices
welded_traingles a QList of AMFTriangle pointers that will contain AMF-formatted triangles that refer to the re-indexed vertices in welded_vertices
Returns:
true if welding was successful
