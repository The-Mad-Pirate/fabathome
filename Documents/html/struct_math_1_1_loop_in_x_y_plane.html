Detailed Description

Stores a set of points that defines a loop. The loop must not be self- intersecting, and its direction matters. A loop that is counter-clockwise when viewed from above on the +Z axis defines an "outer" boundary such that the area within the loop is considered the inside. Winding points in the opposite order defines a loop whose inside contains everything except what is within the loop's boundaries. In this way, taking the union of several loops can define disjoint areas and areas with inner holes.

void Math::LoopInXYPlane::expand ( Float  amount,
QVector< LoopInXYPlane > *  expanded_loops 
) const

Creates a new set of zero or more loops that are displaced by 'amount' from this loop. For some loop shapes, this can cause a single loop to generate multiple loops (by contracting a bean shape, for example). Other times, a high displacement can cause no valid loops to be found at all.

Note:
BE SURE before invoking this method that the loop has been simplified! If at all possible, use the simplifyAndExpand function to call both.
