bool requiresSupportMaterial ( const QList< AMFRegion * >  printing_regions  ) 

Determines whether or not to print the given set of regions requires support material. This calculation should err on the side of caution, returning only false positives. Generally, support material is required whenever overhangs occur. However, material-specific attributes may allow shallow overhangs to be built without support material; those are not accounted for in this approximation.

Parameters:
printing_regions All regions that are going to be printed
Returns:
Whether or not the printing regions require supporting material
