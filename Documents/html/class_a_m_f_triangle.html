bool AMFTriangle::isDegenerate (  )  const

Determines whether or not this AMF triangle is degenerate An AMF triangle is degenerate if its vertices do not determine a triangle. That is, if any two of the triangle's vertex indices are equal the triangle is degenerate.

Returns:
true if this AMF triangle is degenerate, false otherwise
