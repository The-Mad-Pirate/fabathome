Detailed Description

Turns a set of slices into a set of paths with certain properties that will cover those slices in material when printed.

This is a base class whose pathing algorithm is virtual so that a subclass can implement one of many algorithms to execute pathing.

This class is not intended for direct use; instead, use PatherInterface.

See also:
PatherInterface
